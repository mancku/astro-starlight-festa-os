## Què és Python?

Python és un llenguatge de programació d'alt nivell i de propòsit general, conegut per la seva simplicitat i llegibilitat. Creat per Guido van Rossum i llançat per primera vegada el 1991, Python ha guanyat popularitat gràcies a la seva facilitat d'ús, una àmplia comunitat de desenvolupadors, i la seva versatilitat en àmbits com el desenvolupament web, l'anàlisi de dades, la intel·ligència artificial, la ciència de dades i moltes altres àrees.

### Característiques de Python:

- **Simplicitat**: La seva sintaxi clara i concisa el fa fàcil d'aprendre, especialment per a principiants.
- **Llenguatge interpretat**: No necessita compilar-se, fet que accelera el desenvolupament i la prova de codi.
- **Versatilitat**: Python s'utilitza en una gran varietat de camps com el desenvolupament web, la ciència de dades, l'aprenentatge automàtic, l'automatització, etc.
- **Gran comunitat**: Existeixen milers de biblioteques i eines desenvolupades per la comunitat, la qual cosa el fa molt extensible.

## Exemple de Codi en Python

Aquí tens un exemple simple d'un programa que demana a l'usuari el seu nom i després li dóna la benvinguda:

```python
# Demana el nom a l'usuari
nom = input("Quin és el teu nom? ")

# Mostra un missatge de benvinguda
print(f"Hola, {nom}! Benvingut a Python!")
```
